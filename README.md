# Purvins_UnityUI
Unity Ieskaite

Izveidota Minecraft aplikācija kurā ir Main menu un divu tēlu pielāgošana un apģērbšana

Iekļauts - 
    Galvenā izvēlnes scene
    Pielāgošanas scene
    Vārda un gada ieraksti tēlam, aprēķināsana
    Lieluma maiņa
    Bruņu "uzvilkšana"
    Mūzika, skaņas efekti

Papildus funkcijas -
    Velkot bruņas uz kāda tēla tās "pievelk pareizajā vietā" un mainot tēla izmēru mainās arī bruņu izmērs.
